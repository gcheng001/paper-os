# Reveal.js Slide 格式示例

> 来源：《AI高质量论文写作法》4答辩.md，固化日期2026-06-11
> 示例主题：量子纠缠（书中演示用例）

## AI生成→Markdown Slides 工作流

### 第一步：GPT-4生成讲解内容

提示词：
```
请查询最新资料，用小学生也能听懂的语言，详细、生动地讲解以下内容：
[论文内容概要]
```

### 第二步：转换为Markdown Slide格式

提示词：
```
请将以下内容转换为 Markdown revealjs slide 格式。
要求：
- 每段对应一页slide
- 每页不超过6项内容
- 数学公式用LaTeX格式
```

### 第三步：导入Gamma

1. 打开 Gamma（gamma.app）
2. 选择 Text transform 模式
3. 粘贴 Markdown 内容
4. 设置：简体中文 + 12张上限
5. Gamma自动配图、排版
6. 可使用聚光灯模式演示

## Markdown Revealjs 格式规范

```markdown
---
title: "论文标题"
format: revealjs
---

## 研究问题

一句话说清楚

---

## 价值

- 实践意义
- 理论贡献

---

## 必要性

![](vosviewer-output.png)

- 研究空白说明

---

## 研究设计

- 数据：来源和规模
- 方法：核心方法
- 分析：关键步骤

---

## 讨论

- 发现1：与常识相悖的结果
- 发现2：理论意义
- 发现3：实践启示

---

## 限制与展望

- 局限1：客观条件
- 展望1：未来方向
```

## 注意事项

- 每页控制在20字以内（不含图表）
- 脱稿讲述，PPT只是提示卡
- 数学公式用 `$...$` 或 `$$...$$` LaTeX格式
- 图表确保美观正确，标注清晰
