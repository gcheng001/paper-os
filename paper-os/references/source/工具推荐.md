# 一、选题

# 1. Perplexity

AI 检索工具，可访问 Semantic Scholar、arXiv 等学术数据库，支持深度研究和反思式工作流。 

使用技巧: 

# ( 1 )从宽泛主题中挖掘研究方向

- 输入一个宽泛的研究主题; 

- 让 AI 扮演 “项目负责人”，要求它列出组建团队需要招募哪些研究方向的人； 

- AI 会给出若干具体方向，每个方向都附有近期高水平研究佐证。 

# ( 2 ) 生成候选选题

- 选定一个方向后，让 AI 扮演该领域专家； 

- 要求它“从你的独特视角出发，提出3个高水平、新颖的研究问题”； 

- 关键指令：“确保该问题与你的研究方向一致，且不能与你能找到的任何近期高水平英文研究相重叠”； 

- 要求 AI 对每个问题在创新性、重要性、可行性三个维度上打分。 

# ( 3 ) 验证选题新颖性

- 让 AI 扮演 “大学图书馆参考咨询馆员”； 

- 关键指令：“在高水平英文文献中找寻与用户题目最为相关的篇目，判定重合程度，指出差异，说明差异的研究价值”； 

- AI会找出相似文献，分析你的选题和已有研究的区别在哪里，这个区别是否值得研究。 

# 2. Elicit

基于 GPT 模型开发的文献推荐工具，可自动抽取文献信息、筛选文献、生成结构化表格。 

使用技巧: 

# ( 1 ) 智能文献筛选

- 输入研究问题，Elicit 自动搜索相关文献； 

- 进入筛选环节，Elicit 根据研究主题自动生成过滤条件； 

- 对每篇文献给出筛选判断及详细理由。 

# ( 2 ) 信息抽取与结构化

- 自动抽取研究设计类型、参与者特征、干预细节、关键发现等; 

- 生成结构化表格，点击任一单元格可跳转原文高亮位置； 

- 支持增量处理，新增文献时只处理新增部分，结果自动更新。 

# ( 3 ) 生成系统文献回顾报告

- 一键生成报告，报告包含总体结果、分项数据汇总、证据支持等； 

- 拥有互动功能，点击结论即可查看所有支持证据，相关部分高亮显示。 

# 二、文献

# 1. 文献计量工具

# ( 1 ) VOSviewer——入门首选

- 从 CNKI、Web of Science 导出文献数据集; 

- 导入 VOSviewer，生成作者合著网络图； 

- 生成关键词共现网络图，快速识别研究热点和冷门方向。 

# (2) Biblioshiny——统计功能强大

- 基于 R 语言，可生成词云、词汇地图 

- 特色功能 “主题地图”：以横轴代表中心度、纵轴代表密度，将主题分为四类—— 

- 第一象限：既重要又有良好发展 

- 第二象限：有发展但不重要 

- 第三象限：边缘主题（可能刚涌现或即将消失） 

- 第四象限：重要但未获良好发展（基础概念） 

# 2. Explainpaper

上传文献，高亮标注某一个词语、短语或者段落，AI 就会自动解读它们。 

# 使用技巧:

- 上传文献 PDF，选中任何不理解的术语或段落； 

- AI 自动给出通俗解释，可继续追问； 

- 综合信息来源包括论文上下文和模型训练语料，即使论文中未解释的术语也能正确识别。 

# 3. Zotero

一款文献管理工具。 

使用技巧: 

# ( 1 ) 一键抓取

插件系统支持各种文献库和网页的一键抓取元信息功能。 

# ( 2 ) PDF 自动重命名

根据元信息（作者、年份、期刊）自动重命名 PDF 文件。 

# ( 3 ) 阅读与批注

Zotero 7 新增 EPUB 和网页快照阅读支持，支持高亮、下划线、手写标注。 

# 三、写作

# 1. 幕布

写作大纲工具。 

使用技巧: 

- 跨平台记录，随时捕捉灵感； 

- 一键移动大纲节点顺序，快速迭代结构； 

- 一键将大纲列表变成思维导图，可视化检视逻辑； 

- 随时聚焦某个章节，开展头脑风暴。 

# 2. Heptabase

卡片笔记与白板拼接，将写作任务拆解为小单元（卡片），在白板上拼图式组合。 

使用技巧: 

# ( 1 ) 输入阶段

用语音输入或手写笔记快速记录想法，并存为卡片。 

# ( 2 ) 组织阶段

- 将卡片拖拽到白板上; 

- 调整顺序、分组、连线，确定讲述的线性步骤； 

- 粒度太粗的卡片直接拖拽拆分，原位置保留新卡片链接。 

# ( 3 ) 输出阶段

- 按顺序将卡片拷贝到写作工具中; 

- 添加承上启下的连接部分; 

- 补充缺失内容，完成格式调整和润色。 

# 3.Quarto

学术写作专业工具。 

使用技巧: 

- 图表自动编号; 

- 章节交叉索引; 

- 参考文献列表自动生成。

# 2. 提示词核心框架

定义角色：如“你是一名资深审稿人”“你是一名大学图书馆参考咨询馆员”。 

描述目标：如 “请检查以下段落的逻辑漏洞”。 

提供背景：如“本文的研究方法是问卷调查，样本量为 

7034”。 

给出示例：如“参考以下格式输出：……”。 

加入思维链：如 “一步步思考” “请先分析再得出结论”。 

注意：同一提示词在不同模型上效果差异很大。提示词必须与你所使用的模型相匹配。

# 五、答辩 PPT 制作

# Gamma

AI 一键生成 PPT。 

使用流程: 

# ( 1 ) 准备内容

用 GPT 生成讲解内容。 

# ( 2 ) 转换大纲

让 GPT 将内容转换成 Markdown revealjs slide 大纲。 

- 提示词：“把上面的回答，每个段落做成 Markdown revealjs slide 的一页。每页包含标题（‘##’ 二级标题）和页内无序号列表。每一页列表内容不超过 6 项。” 

# ( 3 ) 生成 PPT

- 进入 Gamma，选择 “Text transform”； 

- 设置幻灯片数量（建议 12 页左右）； 

- 设定语言为简体中文; 

- 粘贴大纲，点击“生成”； 

- 选择模板风格，Gamma自动配图、排版。 

# ( 4 ) 输出

支持在线演示（聚光灯模式可逐个元素高亮）、导出PPT或PDF文件。 

# 六、英语能力提升

# 1. Immersive Translate

双语阅读。 

使用技巧: 

- 安装插件后，英文文章一键呈现双语对照。 

- 关键原则：先看英文，不懂再看译文。 

- 不要全文依赖翻译，否则英语能力会衰退。 

# 2.Raycast

定制化划词翻译。 

使用技巧：选中单词，使用快捷键，一键获取释义、音标、例句、联想词。 

请记住，AI辅助科研工具的更新换代速度很快，请各位读者在实际使用时不要拘泥于这个小册子，可以根据现实情况举一反三，找到此时此刻更好的工具。
