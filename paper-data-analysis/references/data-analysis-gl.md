# 路径 A：智谱清言 GLM4 零代码数据分析

> 本文件固化智谱清言 GLM4 零代码数据分析的标准流程与提示词模板。
> 方法论来源：Get 笔记《国产大语言模型辅助科研：写作润色与数据分析实操教程》（王树义课程）

---

## 适用对象

- **无编程基础研究者**：无需写 Python/R 代码，自然语言指令完成
- **快速探索性分析**：拿到数据后快速看分布、做对比、出图表
- **论文结果章初稿**：产出可直接用于结果章的图表 + 数据表 + 解读（**须 S6 复核**）

## 工具

**智谱清言 GLM4**（数据分析功能）

- 访问：chatglm.cn
- 输入：上传 CSV 文件
- 交互：自然语言指令，模型自动写代码执行并返回结果

## 标准分析指令模板（自然语言）

### 1. 描述性统计

```
统计 [字段] 的分布并可视化。

要求：
- 给出均值、中位数、标准差、最大值、最小值
- 绘制分布图（直方图/箱线图）
- 对分布形态做简要文字说明
```

### 2. 分组对比

```
对比 [分组字段] 下 [指标] 的平均值/差异并可视化。

要求：
- 按分组给出指标的描述统计
- 绘制分组对比图（柱状图/分组箱线图）
- 做差异显著性检验（如 t 检验/方差分析），给出 p 值
- 用自然语言解读差异含义
```

### 3. 加权计算

```
按 [权重字段] 计算 [指标] 的总评。

要求：
- 明确权重口径（如权重总和为 100 或为 1）
- 给出加权后的总评排名
- 对比加权前后排名变化
- 用自然语言解读
```

### 4. 趋势 / 进退步分析

```
统计 [指标] 从 [时间1] 到 [时间2] 的变化，按变化幅度排名。

要求：
- 计算每个对象在两个时间点的指标差值与变化百分比
- 按变化幅度降序排名，绘制排名图
- 区分"进步"与"退步"两类
- 列出进步最大和退步最大的前 5 名
- 用自然语言解读
```

### 5. 建模（回归 / 分类）

```
基于 [特征字段列表] 预测 [目标字段]，给出模型准确率和关键特征。

要求：
- 选择合适的模型（分类/回归），说明选择理由
- 划分训练集/测试集（如 8:2），报告测试集准确率/R²
- 给出特征重要性排名
- 对模型结果做自然语言解读，指出哪些特征最关键
- 提示模型的局限性
```

## 提示词设计原则

| 原则 | 说明 |
|------|------|
| **给模型清晰角色** | 如"你是一位数据分析专家" |
| **明确指标口径** | 不留模糊空间，如"权重总和为 100"而非"按权重加权" |
| **要求可视化** | 每个分析都明确要求绘图（指定图表类型） |
| **要求二次解读** | 不要只给数字，必须用自然语言解释含义 |
| **拆分多轮对话** | 复杂分析拆成多步，每步确认后再进下一步 |

## 输出要求

每个分析产出三件套（**可直接用于论文结果章，但须 S6 复核**）：

| 产出 | 形式 | 论文用途 |
|------|------|----------|
| **可视化图表** | PNG/SVG | 论文插图（须 S6 复核 + 人工美化） |
| **数据表** | Markdown 表格 | 论文结果表（须 S6 复核数字） |
| **文字解读** | 自然语言段落 | 论文结果章文字初稿（须 S6 复核 + 去 AI 味） |

## 风险提示

### 1. GLM4 可能产生幻觉数据

- **数据刻板印象**：模型可能基于训练数据脑补统计结果
- **幻觉统计**：模型可能输出实际数据中不存在的数字
- **图表失真**：图表可能存在坐标轴/标签/数值错误

**对策**：所有数字、图表、结论进入论文正文前**必须 S6 复核**，由 S6 通过原始数据源核对（重新计算或人工核对）。

### 2. 大模型对符号平等处理

GLM4 等大模型对各类符号（标点、引文标记、公式符号）平等处理，可顺便利用：

- 整理参考文献格式（APA / GB-T 7714）
- 统一全文标点
- 公式符号规范化

### 3. 数据隐私

- 敏感数据（含个人信息、商业机密）上传前**脱敏**
- 涉密数据使用本地部署版本，不上传云端

## 实操示例（参考 Get 笔记演示）

**场景**：虚拟小学成绩单数据分析

| 步骤 | 指令 | 产出 |
|------|------|------|
| 1 | 上传 `虚拟小学成绩单.csv` | GLM4 自动识别字段 |
| 2 | "统计男女生人数并可视化" | 性别分布饼图 + 数据表 |
| 3 | "统计学生姓氏分布，按人数排名前 10" | 姓氏排名柱状图 |
| 4 | "按平时成绩+期中+期末（3:3:4）计算学期总评，排名前 10" | 加权总评排名表 |
| 5 | "统计成绩分布（优/良/中/差），可视化" | 成绩分布直方图 |
| 6 | "对比男女生总评均值，做显著性检验" | 性别对比箱线图 + t 检验 p 值 |
| 7 | "按期中→期末变化幅度排名，列出进步最大和退步最大的前 5 名" | 进退步排名图 + 名单 |

**关键**：以上每一步产出**都须 S6 复核**才能进入论文正文。

## 与路径 B 的协同

若数据存在字段命名不一、单位混乱、计算口径模糊等问题：

1. **先跑路径 B（Codex 脏数据 6 步核查）**，产出 `_修复版.csv`
2. **再跑路径 A**，上传 `_修复版.csv` 进行分析
3. 分析报告在 frontmatter 中注明数据来源为 `_修复版`
